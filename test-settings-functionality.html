This is sample content to demonstrate how settings affect the appearance of the application.

Toggle the settings above to see how they change the visual presentation:

  • Dark Mode: Switches between light and dark themes
  • Theme Color: Changes the primary accent color
  • Font Size: Adjusts text size throughout the interface
  • High Contrast: Increases color contrast for better visibility
  • Compact Mode: Reduces spacing for more content density
  • Reduced Motion: Disables animations and transitions

All settings are automatically saved to localStorage and persist across sessions.
